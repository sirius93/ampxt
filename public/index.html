  • 1555
    Behavioral Cloning using Deep Learning
    Karthikeyan Palanisamy Saurabh Nutan Sohoni

    Machine learning networks used till 2010 behaved just as a processing unit with garbage in - garbage out strategy. Arriving at a significant output required feeding peculiar features of vector (such as raw RGB values, Histogram of Gradients, etc. in case of computer vision applications) of input data. And computational units extracting these features almost couldn't process anything in real time. With Deep learning, raveled features vectors of raw data (raw pixel values in images) can be fed to a network and desired output can be obtained, output being any sort of a hyper parameter. Deep learning models have already being trained, to use them is to just plug and play. In this session, our team will try to drive a car autonomously on a simulator track by clone driving behavior of a human on the same/similar track. This approach can be extended to drive a robot car through a maze in real world with minimal training.
